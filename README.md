# Envíos

Pantalla principal de una app de compras/envíos, inspirada en Amazon, hecha con Flutter.

**Rápido • Seguro • Confiable**

## Contenido
- Logo propio (`assets/images/logo.jpeg`)
- AppBar con logo y búsqueda
- Banner de ofertas
- Categorías
- Lista horizontal de ofertas
- Grid de productos recomendados
- Barra de navegación inferior

## Cómo correrlo

1. Instala Flutter: https://docs.flutter.dev/get-started/install
2. Abre una terminal en esta carpeta y ejecuta:
   ```
   flutter pub get
   flutter run
   ```

## Cómo subirlo a GitHub

```
git init
git add .
git commit -m "Pantalla principal Envíos"
git branch -M main
```
